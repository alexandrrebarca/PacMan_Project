Pac-Man არის კლასიკური თამაში, სადაც მოთამაშე მართავს Pac-Man-ს ლაბირინთში. მისი მიზანია შეაგროვოს ყველა წერტილიდა დააგროვოს მაქსიმალური ქულები,და თავიდან აირიდოს მოჩვენებები.

თამაშის დაწყებისას Pac-Man მოძრაობს ისრების ღილაკებით. ლაბირინთში განთავსებულია ჩვეულებრივი წერტილები წერტილის შეჭმისას მოთამაშე იღებს ქულებს,მოთამაშის მიზანია დააგროვოს ყველა ქულა ისე რომ Ghosts არ შეეხოს.
